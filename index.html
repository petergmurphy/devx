PCT

PCT

Puppet Content Templates (PCT) codify a structure to produce content for any Puppet Product that can be authored by Puppet Product Teams or external users without direct help of the PDK team.

View Docs Install
PRM

PRM

Puppet runtime manager (PRM) is a program that allows for the management of Puppet versions, along with their respective Ruby versions. This is achieved through the use of custom backends (e.g. local or docker) resulting in faster, more efficient functionality when compared to the PDK.
